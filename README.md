# Bolerplate FastAPI

## Requirements

- Python version: see `.python-version`. We recommend using [pyenv](https://github.com/pyenv/pyenv) to manage python versions.

## Setup

- python -m venv venv
- . ./venv/bin/activate
- pip install -r requirements.txt
- pip install -r requirements_dev.txt
- python src/server.py
